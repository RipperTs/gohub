# gohub

## Air 自动重载
### 安装 air
使用以下命令来安装 air ：  
```
GO111MODULE=on  go install github.com/cosmtrek/air@latest  
```

安装成功后使用以下命令检查下：  
```
air -v

  __    _   ___
 / /\  | | | |_)
/_/--\ |_| |_| \_ , built with Go
```

### 使用 air
在项目目录下运行以下命令：
```
air
```

### air 配置信息
我们可以使用 `.air.toml` 文件来配置 `air` 的行为。
